About the Pipeline Dashboard

Free, open-source Bitcoin intelligence — built on structure, not opinion

The Problem

Bitcoin market intelligence is trapped in a gap. It's either too expensive and raw (Glassnode at $39-$800/month — you get charts, you interpret), too noisy and opinionated (Crypto Twitter, YouTube shills), or too specialized (CoinGlass for liquidations only, TradingView for charts only).

Nobody synthesizes across all these data sources into a single, actionable verdict for free.

What We Built

The Pipeline Dashboard is a synthesis layer. It takes data from Binance, Yahoo Finance, FRED, on-chain providers, and Coinglass — then produces one weighted risk synthesis with position-sizing parameters. Not 50 charts to interpret. One risk tier: PROCEED, TIGHTENED, PAUSE, or ABORT.

Philosophy

Physics, not opinion. The dashboard tracks where leverage is concentrated, where liquidity will pull price, and who's positioned which way — then synthesizes that into a risk synthesis. No moon calls, no price targets, no "trust me bro."

Three Principles

  • Structure over price. Most dashboards lead with price. This one leads with structure — where will price be forced to go? — and treats price as the last variable.
  • Synthesis over data. Raw data is useless without interpretation. Every metric is synthesized into a risk parameter, not just displayed.
  • Free and open. MIT-licensed, no paywalls, no tracking, no data collection. The code is on GitHub. Fork it, build your own, verify the logic.

Privacy

This website collects no data. No cookies, no analytics, no tracking pixels. The dashboard runs entirely in your browser — it fetches JSON data files from the server, but no information about you is sent anywhere. Your visit is invisible.

Not Financial Advice

Important: The Pipeline Dashboard is a tool for market analysis, not financial advice. The risk parameters are generated by algorithms that have known failure modes. The judgment is always yours. Never invest more than you can afford to lose.

Technology

Built with plain HTML, CSS, and JavaScript — no React, no bundler, no npm. The data pipeline is Python, running on cron, fetching from free public APIs. Everything is static, served via GitHub Pages. Open index.html in a browser, it works.

Open Source

The entire project is on GitHub under the MIT license. You can:

  • Read the source code and verify the logic
  • Fork it and build your own version
  • Contribute improvements via pull requests
  • Use the data feeds for your own analysis
