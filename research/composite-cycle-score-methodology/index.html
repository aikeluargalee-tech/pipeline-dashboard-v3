Composite Cycle Score: Synthesizing On-Chain Metrics

On-Chain Valuation · 6 min read

Three on-chain metrics — MVRV Z-Score, SOPR, and Puell Multiple — each tell part of the story. The composite cycle score combines them into a single 0-100 number that tells the whole story at a glance.

How It Works

The dashboard's compute_composite_score() function normalizes each metric to a 0-100 scale, then averages the available scores. If only one metric is available (e.g., MVRV but not SOPR), the composite uses just that one. If all three are available, it averages all three.

Normalization Maps

Each metric has its own normalization curve based on historical ranges:

MVRV Z-Score (range: -1 to 7)

  • Negative → 0-20 (accumulation)
  • 0-2 → 20-50 (fair value to moderate bull)
  • 2-4 → 50-80 (overheating)
  • 4+ → 80-100 (cycle top)

SOPR (range: 0.7 to 2.0)

  • 0.6-0.8 → 0-20 (capitulation)
  • 0.8-1.0 → 20-40 (near breakeven)
  • 1.0-1.5 → 40-70 (normal profit-taking)
  • 1.5+ → 70-100 (aggressive selling)

Puell Multiple (range: 0.2 to 6)

  • 0-0.5 → 10-30 (miner capitulation)
  • 0.5-2 → 30-60 (normal)
  • 2-4 → 60-80 (elevated)
  • 4+ → 80-100 (overheated)

Interpreting the Score

  • 0-30: Deep value / accumulation zone. Historically the best time to accumulate.
  • 30-50: Fair value. Normal market conditions.
  • 50-70: Elevated. Bull market in progress but approaching caution zone.
  • 70-100: Overheated. Historical sell zone — reduce exposure.
0.5x weight in synthesis: The composite score feeds into the regime synthesis at 0.5x weight because cycle data is monthly-scale. It provides context but cannot override live structural or derivatives signals.
