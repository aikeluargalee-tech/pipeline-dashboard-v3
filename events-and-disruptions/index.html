Events & Disruptions

Historical Catalog · 35+ Events · 2021–2026

Bitcoin doesn't move in a vacuum. Every major price event has a cause — a macro shock, a structural unwind, a geopolitical rupture, or a systemic failure. This catalog documents 35+ events with their transmission chains (how the event flows through to BTC price) and rhyme triggers (compound conditions that, when matched, signal a similar event may be unfolding).

The rhyme framework isn't about predicting events — it's about recognizing patterns when they recur. Each event carries compound triggers: no single condition fires; two or more must match simultaneously. "No rhyme" is a valid answer.

Sources: Antigravity (Gemini 3 Pro), Perplexity AI, Codex CLI, GetClaw, CoinBureau. Multi-agent research, cross-referenced.

Rhyme Framework — Expiry-FOMC Proximity Rule

Discovered by Perplexity AI cross-referencing every quarterly expiry against the FOMC calendar, 2021-2025. Calibrated by GetClaw for base-rate statistical significance.

TierWindowExpiriesHit RateBase RateSignalVerdict
CRITICAL0-3 days8/2040.0%15.3%2.6×STRONG — real clustering
HIGH4-7 days4/2020.0%15.3%1.3×MODERATE — slight above chance
MODERATE8-14 days3/2015.0%15.3%1.0×NOISE — indistinguishable from chance
LOW15+ days5/2025.0%38.6%0.6×Below base — clustering confirmed

Compound trigger: Quarterly expiry + FOMC within 0-3 days (CRITICAL tier) + elevated OI + spot trend extended.

Key insight: Only the CRITICAL tier carries real predictive weight. HIGH tier is marginal — amplifier only, not standalone trigger. MODERATE tier is noise (at base rate). Witching amplifies existing narratives, it doesn't create new ones.
