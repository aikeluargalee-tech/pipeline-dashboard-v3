Frequently Asked Questions

Everything you need to know about the Pipeline Dashboard

Should I buy Bitcoin right now?

This dashboard doesn't recommend buys or sells. It monitors 5 data layers — macro conditions, market structure, derivatives positioning, on-chain health, and technical context — and outputs a regime risk synthesis. When conditions are clear it shows Conditions Clear. When a human override is active it shows On Hold. The decision is always yours.

Is Bitcoin crashing today?

The dashboard tracks real-time risk parameters — derivatives positioning, liquidation clusters, and funding rate extremes. It flags when conditions that precede crashes are present. It does not predict crashes, but monitors the warning signs.

How often does the dashboard refresh?

The dashboard auto-refreshes every 15 minutes from live data sources including Binance, FRED, and Yahoo Finance. All analytical layers are recalculated on each refresh.

How do I know if Bitcoin is overpriced?

The dashboard includes MVRV Z-score and Puell Multiple — two on-chain valuation metrics used by institutional analysts. MVRV below 1.0 historically indicates undervaluation. MVRV above 4 historically signals cycle tops. The dashboard auto-refreshes every 15 minutes with live readings.

What is Gate0?

Gate0 is the pre-trade risk filter that sits above all other layers. Five independent modules (Black Swan, VIX/SPX, AI-3 Wave, Stablecoins, Session) each monitor a different category of systemic risk. The most restrictive module wins. Gate0 can veto the entire pipeline — if it says ABORT, no other layer matters. Learn more →

What data sources does the dashboard use?

Binance (price, funding, open interest, klines), Yahoo Finance (SPY, QQQ, VIX, DXY), FRED (treasury yields, M2), Coinglass (liquidation heatmaps), and on-chain providers for MVRV, SOPR, and Puell Multiple. All sources are free public APIs.

Is this financial advice?

No. The dashboard is a market analysis tool. Risk parameters are generated by algorithms with known failure modes. The judgment is always yours. Never invest more than you can afford to lose.

What are liquidation magnets?

Leverage clusters that act as price magnets — price tends to move toward areas where liquidations will cascade. The dashboard tracks the nearest above and below clusters and detects regimes: Vice Grip (both within $500), Upside Squeeze (above closer), Downside Sweep (below closer). Learn more →

What is the VAL Absorption setup?

VAL Absorption detects when price tests the Value Area Low with declining volume and CVD holding — a pattern suggesting institutional accumulation. It's one of three active setup monitors (alongside Breakout-Retest and Breakdown-Retest) running on the dashboard.

Why is technical analysis deprioritized?

TA is lagging — it describes what already happened. The dashboard treats it as context only, last priority. It cannot generate setups on its own. If your read on BTC starts with the chart, you're already wrong about what matters.

Is the dashboard free?

Yes. Completely free, open-source under the MIT license. No paywalls, no tracking, no data collection. The code is on GitHub.

What is the L-1 Manual Gate?

The L-1 Manual Macro Gate is a human override for geopolitical or macro events that no automated system can ingest — wars, regulatory crackdowns, exchange hacks. When active, it overrides all automated analysis with a PAUSE or ABORT risk tier.

How accurate are the model setups?

Every setup emits a confidence label (HIGH, MEDIUM, LOW) based on empirical hit rates. HIGH requires 70%+ hit rate over 90 days with 30+ trades. If a HIGH label drops below 65% for 30 days, it's auto-downgraded to MEDIUM.

What happens during weekends?

Weekend trading is thinner and derivatives parameters are noisier. The session module automatically downgrades to TIGHTENED on weekends. Treat weekend parameters with extra skepticism.

Can I fork the dashboard?

Yes. The project is MIT-licensed on GitHub. You can fork it, read the source code, verify the logic, and build your own version.
